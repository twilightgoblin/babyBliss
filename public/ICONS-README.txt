PWA ICONS NEEDED
=================

Please create the following icon files:
- icon-192.png (192x192 pixels)
- icon-512.png (512x512 pixels)

You can:
1. Use the generate-icons.html file in the root directory
2. Use an online tool like https://realfavicongenerator.net/
3. Design custom icons with your preferred design tool

For now, the app will use icon.svg as a fallback.

Recommended design:
- Pink to purple gradient background (#ec4899 to #a855f7)
- Baby emoji or custom baby icon
- Rounded corners (128px radius for 512x512)
